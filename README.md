## Heapsort

Example of heapsort in Python.

Usage:

    python heapsort.py c a b

Simulating command line example:

    >>> from heapsort import *
    >>> printSortArgs(heapsort, ['c', 'a', 'b'])
    Sorting from:
        ['c', 'a', 'b']
    to:
        ['a', 'b', 'c']

To test all, run:

    python test.py
